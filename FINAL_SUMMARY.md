## 🆘 COMMON USER ISSUES & SOLUTIONS

### "Blank Screen When I Open It"
→ Wait 10-15 seconds, it's loading. Don't close it!

### "It Says USB Device Not Found"
→ Check USB cable connection and make sure device is powered on

### "Can't Find My COM Port"
→ Check Device Manager in Windows. If not there, install CH340 driver.

### "WiFi Connection Won't Work"
→ Make sure device is in WiFi mode and IP is 192.168.4.1

### "Application Keeps Crashing"
→ Restart the application and device. Check Event Viewer for errors.
